# unbored

A lively and bright, fully free web page that helps users cure boredom with:
- predefined and generated activities/games
- situation-based forms
- dedicated quizzes section
- random pickers with filters
- forms to add new activities and quizzes

## Run locally

Open `/home/runner/work/unbored/unbored/index.html` directly in a browser,
or serve it with:

```bash
cd /home/runner/work/unbored/unbored
python3 -m http.server 8000
```
